# DNS Over HTTPS Proxy (Vercel Edition)
Unblock unavailable DoH proxies using Vercel.

## Disclaimer
Do not misuse, I'm not responsible for your usage.

## Usage
Just set the URL Vercel gives to your app as DNS over HTTPS server in your browser settings.

## Deploying on Vercel
To deploy with Vercel, press the button below and complete the setup on Vercel.
